# 🚀 ADA COMICS 🚀

URL: https://marvel-comics-ada.netlify.app/

------------
### Descripción

Este proyecto es un buscador de comics que permite explorar cómics y personajes del universo Marvel, utilizando la API de Marvel para obtener datos.

![image](https://github.com/user-attachments/assets/0d166da8-97e5-4cda-8846-75579b1e8127)


------------

### Funcionalidades:
**Buscar comics y personajes.** Podras buscar comics por titulo o por nombre de personaje.

**Ordenar resultados.** Podras ordenar los resultados por orden alfabetico y tambien en  orden ascendente y descendente.

**Detalle de comic.** Podras acceder a los detalles de los comics y ver titulo, fecha de pucblicacion, escritores y descripcion.

**Detalle de personaje.**  Podras acceder a los detalles de los personajes y ver nombre de personaje y descripcion.

**Navegar a paginas exactas.** Podras elegir a que pagina de los resultados visitar.

------------

### Objetivo y tecnologias utilizadas:
- Objetivo: Desarrollar una aplicación web que permita buscar, listar y visualizar información detallada sobre cómics y personajes utilizando una API REST, con capacidad de personalizar las búsquedas y ordenar los resultados. La aplicación será responsive, usará un preprocesador CSS, incluirá paginación, y estará desplegada en una plataforma accesible. Además, se gestionará el desarrollo a través de ramas y pull requests.


- **HTML5** 
- **CSS:**

- **JavaScript**
- **API de marvel**
